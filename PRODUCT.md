# Product: Spellbook (v0)

## Scope (v0)
- macOS Spellbook app scaffold using SwiftUI and MVVM.
- Basic navigation shell with placeholder views.
- Local-first focus (no networking yet).

## Non-Goals (v0)
- No sync or cloud accounts.
- No authentication or user profiles.
- No advanced filtering, search, or tagging.
- No spell data ingestion or API integration.
- No persistence layer implementation yet (SwiftData/Core Data later).
- No cross-platform (iOS/iPadOS) targets.
